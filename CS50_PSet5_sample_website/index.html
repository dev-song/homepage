Modern webpages are made up of 3 core languages

- HTML, CSS, JavaScript -

HTML

The standard with which webpages are written.

HTML is resposible for organizing the layout and structure of a webpage.

CSS

An acronym for 'Cascading Style Sheets'.

CSS takes charge of styling, or changing the aesthetics of a webpage.

JavaScript

Webpages can be more dynamic with JavaScript.

With its similarity to C, JavaScript is interpreted from top to bottom.
